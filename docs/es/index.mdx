# Documentación de la API Blockchain

Bienvenido a nuestra documentación exhaustiva de la API Blockchain. Nuestra plataforma proporciona acceso fiable y seguro a múltiples redes blockchain a través de endpoints API estandarizados.

## Blockchains Compatibles

<Link to="/docs/bitcoin">Bitcoin</Link> - La blockchain de criptomoneda original que ofrece transacciones seguras y descentralizadas con soporte para mainnet.
<Link to="/docs/ethereum">Ethereum</Link> - Plataforma de contratos inteligentes que soporta tanto mainnet como testnet Sepolia con conexiones HTTPS y WebSocket.
<Link to="/docs/bsc">BNB Smart Chain</Link> - Blockchain rápida y de bajo costo para DeFi y dApps con entornos mainnet y testnet.
<Link to="/docs/solana">Solana</Link> - Blockchain de alto rendimiento con procesamiento rápido de transacciones en mainnet y devnet.
<Link to="/docs/ton">TON</Link> - Plataforma multi-blockchain escalable con integración de TON Center.

## Primeros Pasos

Nuestra API proporciona acceso uniforme a todas las blockchains compatibles, permitiendo a los desarrolladores:

- Consultar datos de la blockchain
- Enviar transacciones
- Monitorear eventos de la red
- Interactuar con contratos inteligentes

Cada sección de blockchain contiene documentación detallada sobre los métodos disponibles, formatos de solicitud y ejemplos de respuesta.

## Métodos de Conexión

Nuestra API soporta múltiples métodos de transporte:
- **HTTPS** - Disponible para todas las cadenas
- **WebSocket** - Disponible para Ethereum, BNB Smart Chain, Solana y TON

## Opciones de Acceso Gratuito

Ofrecemos acceso de nivel gratuito a blockchains seleccionadas para ayudarte a comenzar:
- Bitcoin
- Ethereum

Explora nuestra documentación para aprender cómo integrar estas potentes redes blockchain en tus aplicaciones.
