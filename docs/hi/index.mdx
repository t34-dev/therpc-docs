# ब्लॉकचेन API दस्तावेज़ीकरण

हमारे व्यापक ब्लॉकचेन API दस्तावेज़ीकरण में आपका स्वागत है। हमारा प्लेटफॉर्म मानकीकृत API एंडपॉइंट्स के माध्यम से कई ब्लॉकचेन नेटवर्क तक विश्वसनीय और सुरक्षित पहुंच प्रदान करता है।

## समर्थित ब्लॉकचेन

<Link to="/docs/bitcoin">Bitcoin</Link> - मुख्य क्रिप्टोकरेंसी ब्लॉकचेन जो मेननेट समर्थन के साथ सुरक्षित, विकेंद्रीकृत लेनदेन प्रदान करता है।
<Link to="/docs/ethereum">Ethereum</Link> - स्मार्ट कॉन्ट्रैक्ट प्लेटफॉर्म जो HTTPS और WebSocket कनेक्शन के साथ मेननेट और Sepolia टेस्टनेट दोनों का समर्थन करता है।
<Link to="/docs/bsc">BNB Smart Chain</Link> - मेननेट और टेस्टनेट वातावरण के साथ DeFi और dApps के लिए तेज़ और कम लागत वाला ब्लॉकचेन।
<Link to="/docs/solana">Solana</Link> - मेननेट और डेवनेट पर तेज़ लेनदेन प्रसंस्करण के साथ उच्च-प्रदर्शन ब्लॉकचेन।
<Link to="/docs/ton">TON</Link> - TON सेंटर एकीकरण के साथ स्केलेबल मल्टी-ब्लॉकचेन प्लेटफॉर्म।

## आरंभ करना

हमारा API सभी समर्थित ब्लॉकचेन में एक समान पहुंच प्रदान करता है, जिससे डेवलपर्स को:

- ब्लॉकचेन डेटा क्वेरी कर सकते हैं
- लेनदेन सबमिट कर सकते हैं
- नेटवर्क इवेंट्स की निगरानी कर सकते हैं
- स्मार्ट कॉन्ट्रैक्ट्स के साथ इंटरैक्ट कर सकते हैं

प्रत्येक ब्लॉकचेन अनुभाग में उपलब्ध मेथड्स, अनुरोध प्रारूपों और प्रतिक्रिया उदाहरणों पर विस्तृत दस्तावेज़ीकरण है।

## कनेक्शन विधियां

हमारा API कई परिवहन विधियों का समर्थन करता है:
- **HTTPS** - सभी चेन के लिए उपलब्ध
- **WebSocket** - Ethereum, BNB Smart Chain, Solana और TON के लिए उपलब्ध

## मुफ्त एक्सेस विकल्प

हम आपको शुरू करने में मदद के लिए चुनिंदा ब्लॉकचेन तक मुफ्त टियर एक्सेस प्रदान करते हैं:
- Bitcoin
- Ethereum

अपने एप्लिकेशन में इन शक्तिशाली ब्लॉकचेन नेटवर्क को एकीकृत करने के तरीके के बारे में जानने के लिए हमारे दस्तावेज़ीकरण का अन्वेषण करें।
