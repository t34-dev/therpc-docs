# 基本操作

本指南涵盖了使用 TheRPC API 在以太坊区块链上执行的最常见操作。

## # 检索基本信息

### 获取最新区块号

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_blockNumber" title="获取以太坊中的当前区块号">eth_blockNumber</Link>。

### 检查账户余额

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    "latest"
  ],
  "id": 1
}
```

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_getBalance" title="获取以太坊地址的以太币余额">eth_getBalance</Link>。

## # 区块信息

### 通过编号获取区块

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockByNumber",
  "params": [
    "0xE81224",
    false
  ],
  "id": 1
}
```

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_getBlockByNumber" title="通过区块编号获取区块信息">eth_getBlockByNumber</Link>。

### 通过哈希获取区块

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getBlockByHash",
  "params": [
    "0xdc0818cf78f21a8e70579cb46a43643f78291264dda342ae31049421c82d21ae",
    false
  ],
  "id": 1
}
```

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_getBlockByHash" title="通过区块哈希获取区块信息">eth_getBlockByHash</Link>。

## # 智能合约交互

### 调用合约方法

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
      "data": "0x70a08231000000000000000000000000742d35Cc6634C0532925a3b844Bc454e4438f44e"
    },
    "latest"
  ],
  "id": 1
}
```

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_call" title="无需交易执行智能合约视图函数">eth_call</Link>。

### 获取合约代码

更多详情请参阅 <Link to="/docs/ethereum/core-methods/eth_getCode" title="获取地址处部署的字节码">eth_getCode</Link>。
