# बुनियादी संचालन

यह गाइड TheRPC API का उपयोग करके Ethereum ब्लॉकचेन के साथ आप जो सबसे आम संचालन करेंगे, उन्हें कवर करती है।

## # बुनियादी जानकारी प्राप्त करना

### नवीनतम ब्लॉक नंबर प्राप्त करें

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

अधिक जानकारी के लिए <Link to="/docs/ethereum/core-methods/eth_blockNumber" title="Get the current block number in Ethereum">eth_blockNumber</Link> देखें।

### खाता शेष जांचें

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    "latest"
  ],
  "id": 1
}
```

अधिक जानकारी के लिए <Link to="/docs/ethereum/core-methods/eth_getBalance" title="Get the Ether balance for an Ethereum address">eth_getBalance</Link> देखें।

## # स्मार्ट कॉन्ट्रैक्ट इंटरैक्शन

### कॉन्ट्रैक्ट मेथड कॉल करें

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
      "data": "0x70a08231000000000000000000000000742d35Cc6634C0532925a3b844Bc454e4438f44e"
    },
    "latest"
  ],
  "id": 1
}
```

अधिक जानकारी के लिए <Link to="/docs/ethereum/core-methods/eth_call" title="Execute smart contract view function without transaction">eth_call</Link> देखें।

### कॉन्ट्रैक्ट कोड प्राप्त करें

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getCode",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
```

अधिक जानकारी के लिए <Link to="/docs/ethereum/core-methods/eth_getCode" title="Get the deployed bytecode at an address">eth_getCode</Link> देखें।
