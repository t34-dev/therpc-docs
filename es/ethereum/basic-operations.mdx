# Operaciones Básicas

Esta guía cubre las operaciones más comunes que realizarás con la blockchain Ethereum usando la API de TheRPC.

## # Recuperación de Información Básica

### Obtener el Número del Último Bloque

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

Ver <Link to="/docs/ethereum/core-methods/eth_blockNumber" title="Obtener el número de bloque actual en Ethereum">eth_blockNumber</Link> para más detalles.

### Verificar Saldo de Cuenta

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_getBalance",
  "params": [
    "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    "latest"
  ],
  "id": 1
}
```

Ver <Link to="/docs/ethereum/core-methods/eth_getBalance" title="Obtener el saldo de Ether para una dirección Ethereum">eth_getBalance</Link> para más detalles.

## # Interacciones con Contratos Inteligentes

### Llamar a Método de Contrato

```javascript
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
      "data": "0x70a08231000000000000000000000000742d35Cc6634C0532925a3b844Bc454e4438f44e"
    },
    "latest"
  ],
  "id": 1
}
```

Ver <Link to="/docs/ethereum/core-methods/eth_call" title="Ejecutar función de visualización de contrato inteligente sin transacción">eth_call</Link> para más detalles.
