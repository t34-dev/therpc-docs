# eth_getUncleCountByBlockHash

`eth_getUncleCountByBlockHash` 方法返回通过区块哈希指定的区块中叔块（ommer区块）的数量。叔块是有效的区块，它们未被包含在主链中，但在以太坊中仍被引用并获得部分奖励。

## 使用场景

- 区块浏览器统计
- 挖矿池分析
- 网络健康监控
- 区块验证确认
- 共识算法研究
- 叔块率分析
- 历史区块数据收集
- 挖矿奖励计算

## 方法详情

此方法返回通过其哈希标识的特定区块中的叔块数量。

<ApiMethodBlock
	chain="ethereum"
	scheme={{
		method: 'eth_getUncleCountByBlockHash',
		id: 1,
		jsonrpc: '2.0',
	}}
	fields={[
		{
			id: 'blockHash',
			required: true,
			type: 'string',
			desc: '区块的哈希',
			value: '0xb3b20624f8f0f86eb50dd04688409e5cea4bd02d700bf6e79e9384d47d6a5a35',
		},
	]}
	returns={[
		{
			id: 'result',
			type: 'string',
			desc: '此区块中叔块数量的整数（十六进制）',
		},
	]}
	getParams={(fields) => [fields?.blockHash || null]}
/>

### 响应示例

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1" // 此区块中有1个叔块
}
```
