# SDK和工具概述

本指南概述了跨不同编程语言与以太坊交互的最流行库和工具。选择最适合您的技术栈和需求的工具。

## 语言支持

我们为各种编程语言和平台提供全面支持：

- <Link to="/docs/ethereum/tools/curl" title="与以太坊节点进行直接HTTP API交互的示例">HTTP/Curl</Link> 用于直接API交互
- <Link to="/docs/ethereum/tools/javascript-typescript" title="用于以太坊开发的JavaScript和TypeScript库指南">JavaScript/TypeScript</Link> 用于Web和Node.js开发
- <Link to="/docs/ethereum/tools/python" title="用于以太坊区块链开发的Python库指南">Python</Link> 用于后端和科学计算
- <Link to="/docs/ethereum/tools/java-kotlin" title="用于基于JVM的以太坊开发的Java和Kotlin库指南">Java/Kotlin</Link> 用于企业和Android开发
- <Link to="/docs/ethereum/tools/swift" title="用于iOS和macOS以太坊开发的Swift库指南">Swift</Link> 用于iOS和macOS开发
- <Link to="/docs/ethereum/tools/dart-flutter" title="用于跨平台以太坊应用的Dart和Flutter工具指南">Dart/Flutter</Link> 用于跨平台移动开发
- <Link to="/docs/ethereum/tools/csharp" title="用于以太坊开发的C#和.NET库指南">C#</Link> 用于.NET生态系统
- <Link to="/docs/ethereum/tools/php" title="用于以太坊开发的PHP库和工具指南">PHP</Link> 用于Web开发

- <Link to="/docs/ethereum/tools/rust" title="用于以太坊开发的Rust库和工具指南">Rust</Link> 用于系统编程

每个语言部分包括推荐的库、代码示例和以太坊开发的最佳实践。
