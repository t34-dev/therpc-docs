# 区块链 API 文档

欢迎阅读我们全面的区块链 API 文档。我们的平台通过标准化的 API 端点提供对多个区块链网络的可靠和安全访问。

## 支持的区块链

<Link to="/docs/bitcoin">比特币</Link> - 原始加密货币区块链，提供安全、去中心化的交易，支持主网。
<Link to="/docs/ethereum">以太坊</Link> - 智能合约平台，支持主网和 Sepolia 测试网，提供 HTTPS 和 WebSocket 连接。
<Link to="/docs/bsc">BNB Smart Chain</Link> - 为 DeFi 和 dApps 提供快速、低成本的区块链，支持主网和测试网环境。
<Link to="/docs/solana">Solana</Link> - 高性能区块链，在主网和开发网络上提供快速交易处理。
<Link to="/docs/ton">TON</Link> - 可扩展的多区块链平台，集成 TON Center。

## 入门指南

我们的 API 提供对所有支持区块链的统一访问，使开发者能够：

- 查询区块链数据
- 提交交易
- 监控网络事件
- 与智能合约交互

每个区块链部分都包含有关可用方法、请求格式和响应示例的详细文档。

## 连接方法

我们的 API 支持多种传输方法：
- **HTTPS** - 适用于所有链
- **WebSocket** - 适用于以太坊、BNB Smart Chain、Solana 和 TON

- 以太坊

浏览我们的文档，了解如何将这些强大的区块链网络集成到您的应用程序中。
