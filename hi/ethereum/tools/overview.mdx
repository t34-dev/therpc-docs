# SDK और टूल्स अवलोकन

यह गाइड विभिन्न प्रोग्रामिंग भाषाओं में Ethereum के साथ इंटरैक्ट करने के लिए सबसे लोकप्रिय लाइब्रेरीज़ और टूल्स का अवलोकन प्रदान करता है। अपने टेक स्टैक और आवश्यकताओं के अनुसार सबसे उपयुक्त टूल्स चुनें।

## भाषा समर्थन

हम विभिन्न प्रोग्रामिंग भाषाओं और प्लेटफॉर्म के लिए व्यापक समर्थन प्रदान करते हैं:

- <Link to="/docs/ethereum/tools/curl" title="Ethereum नोड्स के साथ सीधे HTTP API इंटरैक्शन के लिए उदाहरण">HTTP/Curl</Link> सीधे API इंटरैक्शन के लिए
- <Link to="/docs/ethereum/tools/javascript-typescript" title="Ethereum विकास के लिए JavaScript और TypeScript लाइब्रेरीज़ का गाइड">JavaScript/TypeScript</Link> वेब और Node.js विकास के लिए
- <Link to="/docs/ethereum/tools/python" title="Ethereum ब्लॉकचेन विकास के लिए Python लाइब्रेरीज़ का गाइड">Python</Link> बैकएंड और वैज्ञानिक कंप्यूटिंग के लिए
- <Link to="/docs/ethereum/tools/java-kotlin" title="JVM-आधारित Ethereum विकास के लिए Java और Kotlin लाइब्रेरीज़ का गाइड">Java/Kotlin</Link> एंटरप्राइज़ और Android विकास के लिए
- <Link to="/docs/ethereum/tools/swift" title="iOS और macOS Ethereum विकास के लिए Swift लाइब्रेरीज़ का गाइड">Swift</Link> iOS और macOS विकास के लिए
- <Link to="/docs/ethereum/tools/dart-flutter" title="क्रॉस-प्लेटफॉर्म Ethereum ऐप्स के लिए Dart और Flutter टूल्स का गाइड">Dart/Flutter</Link> क्रॉस-प्लेटफॉर्म मोबाइल विकास के लिए
- <Link to="/docs/ethereum/tools/csharp" title="Ethereum विकास के लिए C# और .NET लाइब्रेरीज़ का गाइड">C#</Link> .NET इकोसिस्टम के लिए
- <Link to="/docs/ethereum/tools/php" title="Ethereum विकास के लिए PHP लाइब्रेरीज़ और टूल्स का गाइड">PHP</Link> वेब विकास के लिए
- <Link to="/docs/ethereum/tools/go" title="go-ethereum पर ध्यान केंद्रित करते हुए Ethereum के लिए Go लाइब्रेरीज़ का गाइड">Go</Link> उच्च-प्रदर्शन एप्लिकेशन के लिए
- <Link to="/docs/ethereum/tools/rust" title="Ethereum विकास के लिए Rust लाइब्रेरीज़ और टूल्स का गाइड">Rust</Link> सिस्टम प्रोग्रामिंग के लिए
